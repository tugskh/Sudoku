<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Download Xcode
2. Clone the repo
   ```sh
   git clone https://github.com/tugskh/Sudoku.git
   ```
3. Build the project
   ```sh
   xcodebuild Sudoku
   ```
4. For help:
    ```sh
   xcodebuild -help
   ```
